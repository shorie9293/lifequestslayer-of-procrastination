# UI/UX & チュートリアル改善に関する議事録・方針

**日付**: 2026-03-09
**参加者**: ユーザー（主君）, 雅仁, 阿国, 官兵衛, 幸村, 誾千代, 輝子

## 議論の要約
ユーザーより、「既存機能を変えずに、チュートリアルやUIを非常にユーザーフレンドリーに改善してほしい」との要望を受け、エージェント間で改善方針を議論した。

- **阿国 (UI/UX)**: 既存の味気ないUIをリッチにするため、角丸、シャドウ、色彩の整理を提案。
- **官兵衛 (実用性)**: 実用性を落とさないよう、チュートリアルは「次に何をタップすべきか」だけを明示するスポットライト型を提案。
- **幸村 (没入感)**: スポットライトに加えて、案内役キャラクターが熱く語りかける吹き出しスタイルを提案。
- **誾千代 (調整)**: メイン画面は余白を活かした洗練されたUIにしつつ、チュートリアル時のみ案内役の吹き出しを表示する折衷案を提示。

## 決定事項（方針）
1. **チュートリアルのスポットライト・ナビゲーション化**
   - 既存の「Step 0, 1, 2...」というテキスト表示を廃止。
   - 画面全体を半透明で覆い、操作対象のみをハイライト（スポットライト）する。
   - 案内役キャラクター（アイコン＋セリフ）を表示し、直感的な言葉で次の操作を促す。
2. **メインUIのモダン化**
   - 既存機能をそのままに、Home, Guild, Town等の各画面のカード、ボタン、余白、色使いなどを洗練されたモダンなデザイン（角丸、シャドウなど）に調整する。
3. **マイクロインタラクションの追加**
   - タップ時の波紋エフェクトや、フィードバックアニメーションを強化して「触り心地」を向上させる。

## 今後の実装ステップ
- カスタムのオーバーレイウィジェットを作成し、スポットライトと案内役吹き出しを実装。
- 各Screen（`home_screen`, `guild_screen` など）のレイアウト・デザインをリファクタリング。
