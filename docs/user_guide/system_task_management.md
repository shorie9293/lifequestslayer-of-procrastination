# タスク管理・クエストシステム

## 概要
本アプリの中心となるタスク（クエスト）管理システムです。ユーザーは現実のタスクをRPGのクエストに見立てて管理することができます。

## 主な機能
1. **タスクの作成とランク設定**
   - 新規タスク作成時に「Sランク」「Aランク」「Bランク」の難易度を設定できます。
   - サブタスク（小タスク）の追加が可能です。
   - 繰り返し設定（毎日、毎週の特定の曜日）に対応しています。
   - 目標時間（タイムアタック用）の設定ができます。

2. **ギルドと戦場（ステータス管理）**
   - 作成されたタスクはまず「ギルド」に登録されます。ギルド内では自由にタスクの編集・削除が可能です。
   - ギルドからタスクを「受注」することで、タスクは「戦場（アクティブ）」に移動します。
   - キャパシティ（クエスト受注枠）の制限があり、プレイヤーの実力（ランクや冒険者レベル）に応じて同時に受注できるタスク数が決まっています。

3. **タスクの完了**
   - 戦場画面でタスクを完了することで、経験値（EXP）や金貨（コイン）を獲得できます。
   - ジョブや疲労度、所持している称号などによって獲得できる報酬が変動します。
