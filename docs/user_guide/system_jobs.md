# ジョブ（職業）システム

## 概要
プレイヤーは4つの基本ジョブ（職業）を切り替えることができ、それぞれのジョブに応じた特性（パッシブスキル）やアプリのテーマカラーが適用されます。

## ジョブ一覧と特性

1. **戦士 (Warrior)**
   - **特性**: コンボボーナス。連続してタスクを完了することで、経験値にボーナス（コンボ数 × 10）が加算されます。
   - **テーマカラー**: 赤・ダークブラウン系

2. **聖職者 (Cleric)**
   - **特性**: 繰り返しタスクのルーティン管理。完了したデイリー・ウィークリータスクをその日のうちは非表示にし、画面をすっきり保ちます。
   - **テーマカラー**: シアン・深緑系

3. **魔法使い (Wizard)**
   - **特性**: サブタスクの厳格化。タスクに紐づくサブタスクがすべて完了していないと、メインタスクを完了できなくなります（タスクの抜け漏れ防止）。
   - **テーマカラー**: ディープパープル・紺系

4. **冒険者 (Adventurer)**
   - **特性**: クエスト受注枠の拡張。レベルが上がるごとに、同時に受注（戦場に出せる）できるS・A・Bランクのタスク枠が増加します。
   - **テーマカラー**: ブラウン・アースカラー系

## スキルマスタリー
各ジョブのレベルを最大（基本はLv14、冒険者はLv10）まで上げると「マスター」状態となり、そのジョブのスキルを他のジョブについている時でも有効化（アクティブスキルとしてセット）できるようになります。これにより、複数のジョブの特性を組み合わせた効率的なタスク消化が可能になります。
