# 称号・デイリーミッションシステム

## 概要
プレイヤーのモチベーションを長期的に維持するための「称号（アチーブメント）」と、日々の習慣化を促進する「デイリー・ウィークリーミッション」のシステムです。

## ミッション（習慣化ボーナス）
特定数のタスクを期間内に完了すると追加の金貨ボーナスが貰えます。

1. **デイリーミッション**
   - 1日に任意のタスクを合計3つ完了すると「デイリーミッション達成ボーナス」として **+200金貨** を獲得できます。
   
2. **ウィークリーミッション**
   - 1週間のうちにSランクのタスクを1つ完了すると「ウィークリーSランク達成ボーナス」として **+500金貨** を獲得できます。
   - 週替り（月曜日などのリセットタイミング）で進捗がリセットされます。

## 称号システム
達成したタスクの累計数や、特定の難易度の達成回数に応じて、さまざまな称号（タイトル）がアンロックされます。

- **取得条件例**:
  - 『見習い冒険者』: 累計タスク完了数10回以上
  - 『ベテラン』: 累計タスク完了数100回以上
  - 『ゴブリンスレイヤー』: 累計 Bランク完了数50回以上
  - 『エリートハンター』: 累計 Aランク完了数20回以上
  - 『竜殺し』: 累計 Sランク完了数5回以上

- **称号ボーナス**:
  - 獲得した称号は1つだけ装備することが可能です。
  - 称号を装備した状態でタスクをこなすと、タスク完了時に得られるEXP（経験値）に **基礎値の+5%のボーナス** が付きます。
