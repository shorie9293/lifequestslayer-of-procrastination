# 宿屋（回復）システム

## 概要
疲労度システムにより、1日に行えるタスクの量には限界があります。この「宿屋（Inn System）」を利用することで、プレイヤーは所持している金貨（コイン）を支払い、翌日のタスク消化上限を引き上げることができます。これによって、一時的な「がんばり」をシステムがサポートします。

## 宿屋のメニュー
宿屋では回復グレードに応じて以下の3つのメニューを選択できます。上位のベッドを利用するほど、翌日の疲労度制限が大きく緩和されます。

1. **テント (Tent)**
   - **費用**: 50 金貨
   - **効果**: 翌日のタスク消化限界（ペナルティの閾値）が **+2回分** 上乗せされます。
   
2. **普通のベッド (Normal Bed)**
   - **費用**: 200 金貨
   - **効果**: 翌日のタスク消化限界が **+5回分** 上乗せされます。
   
3. **王様のベッド (King's Bed)**
   - **費用**: 1000 金貨
   - **効果**: 翌日のタスク消化限界が **+12回分** 上乗せされます。通常よりも大幅に多くのタスクを、ペナルティなしでこなせるようになります。

## 仕様と制限
- 宿屋は **1日に1回まで** のみ利用可能です。同じ日に複数回休むことはできません（「今日はもう十分休んだ。また明日来な！」というメッセージが表示されます）。
- 効果は「前夜に十分休んだことによる翌日のパフォーマンス向上」というコンセプトで、宿屋を利用した日の**「翌日」**のタスク上限に対してボーナスが適用されます。
