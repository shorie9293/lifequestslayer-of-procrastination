# 報酬・経験値・疲労度システム

## 概要
タスクを完了することで得られる報酬（経験値・金貨）の計算ルールと、プレイヤーの過剰なプレイ（不正防止ややりすぎ防止）を管理する疲労度システムについて説明します。

## 経験値（EXP）とレベルアップ
- **基本報酬**:
  - Sランク: 1000 EXP
  - Aランク: 300 EXP
  - Bランク: 100 EXP
- **レベル計算**:
  - 次のレベルに必要な経験値は `50 * 1.4^(現在レベル-1)` で計算され、レベルが上がるごとに必要経験値が増加します。
- **ボーナス要素**:
  - 「戦士」ジョブのコンボボーナス、または装備している称号（+5%）によって獲得経験値が増加します。

## ゲーミフィケーション要素（コイン・ドロップ）
- **金貨（コイン）**:
  - タスク達成ごとに Sランク 100枚、Aランク 30枚、Bランク 10枚 の金貨が獲得できます。金貨は宿屋や今後実装予定のショップ等で使用します。
- **タイムアタックボーナス**:
  - 事前に設定した目標時間（Target Time）以内にタスクを完了すると、**スピードクリアボーナスとして+50金貨**が追加で支払われます。
- **レアドロップ**:
  - タスク完了時、プレイヤーのレベルに応じた確率（レベル×2%、最大50%）でレアドロップが発生し、獲得金貨が大幅（通常の5倍）に増加します。

## 疲労度システム
- **目的**: 1日に無制限にタスクをこなすこと（不正に報酬を得ること）や、バーンアウト（燃え尽き）を防ぐための仕組みです。
- **制限の仕組み**:
  - 1日に完了したタスクの数が一定の閾値（Threshold）を超えると、得られる報酬（EXPや金貨）にペナルティがかかります。
  - **閾値（警告）**: 5回以上（+宿屋オフセット）タスクを完了すると、報酬が50%（0.5倍）に減少します。
  - **閾値（深刻）**: 10回以上（+宿屋オフセット）タスクを完了すると、報酬が10%（0.1倍）に激減します。
- **リセット**: 日付が変わる（または宿屋を利用する）ことで疲労度がリセットされ、翌日の限界値も更新されます。
